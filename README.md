# Proposta
  A proposta desse projeto é realizar uma adaptação da linguagem de programação python para que possa ser aplicada em processos de segurança na realização de assessments em dispositivos IoT (Internet das Coisas).

# Escopo
  O objetivo é alcançar a proposta acima de maneira a aplicar tal linguagem de programação em pratica. 
  
  Contudo, nosso escopo inicial será:
  - Adaptar o uso de uma linguagem de programação ao objetivo do projeto.
  - Realizar tratativas de possíveis erros de compilação.
  - Aplicar essa linguagem de programação no uso de recursos do S.O (Sistema Operacional), como recursos de rede IPV4.


# Motivação
- Nossas motivações para utilizar a linguagem Python neste trabalho foram:
  - Facilidade de aprendizado: Python é uma linguagem de programação de alto nível que é relativamente fácil de aprender. 
  - Multi-paradigma: Python é uma linguagem multi-paradigma, o que significa que ela suporta diferentes abordagens de programação. 
  - Abundância de bibliotecas: Python possui uma grande quantidade de bibliotecas e frameworks disponíveis para diferentes tarefas, incluindo ciência de dados, machine  learning, desenvolvimento web, automação de tarefas e muito mais. 
  - Comunidade ativa: Python possui uma comunidade de desenvolvedores ativa e colaborativa que contribui para a criação e manutenção de bibliotecas, frameworks e documentação. 
  - Portabilidade: Python é uma linguagem portátil, o que significa que o código escrito em Python pode ser executado em diferentes plataformas. 



# Semântica
- Python é uma linguagem de programação interpretada de alto nível e que suporta múltiplos paradigmas de programação: imperativo, orientado a objetos e funcional.
